## 进度更新规则

阶段状态只能是：

```text
未开始
进行中
阻塞
已完成
暂不投入生产
```

更新“已完成”必须附：

- 修改文件；
- 测试命令和结果；
- 真实联调证据；
- 未解决问题；
- 是否超出范围。

## 估算原则

- 先按可验收纵向切片估算，不按页面数量估算；
- 后端未就绪的阶段不承诺完成日期；
- 发现跨三个以上模块的任务必须重新拆分；
- 任何“顺便重构”单独立项。

## 当前推荐顺序

```text
已完成：F0 -> F1 -> F2 -> F3 -> F4 -> F5 -> F6 -> F7
前端实施循环完成
```

F0～F7 已完成；后续变更按独立需求和生产发布流程执行。
